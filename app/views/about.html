Best of Bot Beta

Best of Bot is an IRC bot/website that tracks the best moments of Daily Tech News Show, an (aptly named) daily tech news podcast hosted by Tom Merritt.

During the Christmas break, Tom does a "best of" show with clips from the previous year. This is obviously a lot of work, so the Best of Bot tracks potential best of moments to help out.

To record a best of moment, join the chat room at irc.dtns.tv, and use the !b identifier (or bangtag, as I'm calling it) followed by a short description. When it sees this it calculates a timestamp relative to the start of the show video. This timestamp is then used to generate an embedded Youtube video that starts at the correct time right here!

Created by Ali Smith (@40_thieves). Source code for the bot and the website frontend are available on Github under a MIT license. If you find a bug, please let me know or on Twitter or via Github issues!
